# Additional Research Materials

This folder contains supporting materials for the research project on:

**LinkedIn organic content strategy for B2B SaaS**

The main raw research lives in:

- `../sources.md`
- `../linkedin-posts/`
- `../youtube-transcripts/`

This `other/` folder is intentionally not a raw-content dump. It contains research support files that explain why the sources are credible, how the collection was done, what the coverage looks like, how source quality was handled, and whether the material is strong enough to support a real playbook later.

## Files

- `expert-selection-rationale.md` - why these 10 experts were chosen.
- `research-coverage-map.md` - which expert covers which part of the topic.
- `collection-notes.md` - technical notes on LinkedIn collection, YouTube transcripts, API usage, and limitations.
- `source-quality-matrix.md` - confidence levels and verification notes for different source types.
- `playbook-readiness-notes.md` - what the research is already strong enough to support later.
- `evaluation-alignment.md` - how the repository maps to the assignment's evaluation criteria.

## Why This Folder Exists

The assignment asks for `/research/other/` as "any additional materials." For this project, I used it to document research judgment rather than add more raw volume.

That means this folder helps answer:

- Are the experts genuinely strong voices?
- Does the source set cover the topic well?
- What tools and APIs were used?
- What limitations appeared during collection?
- Is this material strong enough to support a real playbook later?
